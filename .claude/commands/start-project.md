# Staring a brand Claude development context

## Primary Objective: Understand the project's main purpose and current state to provide appropriate support.

## Perform a detailed analysis of all folders in this project to understand its goals and establish the initial context for development assistance.

### Actions to Perform:

    Project Analysis

        - Detect if a project exists or if a new one should be created.

        - Identify the primary programming language from the existing files. If unclear, ask the user for clarification.

        - Determine the project type (e.g., web app, API, library, CLI tool).

        - Check for existing configuration files to avoid overwriting them.

        - Understand the project's main objective and structure.

    Documentation Generation
    Create essential documentation files or update them:

        - README.md: Provide a project overview, setup instructions, and basic usage examples.

        - CONTRIBUTING.md: Outline guidelines for development and contributions.

        - LICENSE: Propose or create a license file based on user selection.

        - CHANGELOG.md: Establish a file for tracking version history.
